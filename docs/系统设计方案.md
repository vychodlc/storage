# 古楼山竹筷厂生产管理系统 - 设计方案

## 1. 项目概述

### 1.1 项目背景
古楼山竹筷厂是一家将竹丝加工为不同尺寸竹筷的生产企业。需要一套完整的生产管理系统来管理进货、生产、出货、库存、工人、出勤和流水等核心业务流程。

### 1.2 技术选型
- **前端框架**: Next.js 16 (App Router)
- **UI组件库**: Ant Design 5.x
- **Pro组件**: @ant-design/pro-components
- **图表库**: Recharts
- **语言**: TypeScript

### 3.6 进货单 (PurchaseOrder)
```typescript
interface PurchaseOrder {
  id: string;
  orderNo: string;        // 订单号
  orderDate: string;      // 单据日期
  supplierId: string;     // 供应商ID
  supplierName: string;   // 供应商名称

  driverName: string;     // 司机姓名
  items: PurchaseItem[];  // 进货明细
  totalAmount: number;    // 总金额
  paidAmount: number;     // 已付金额
  status: 'pending' | 'partial' | 'completed';  // 付款状态
  payments: Payment[];    // 付款记录
  remark: string;         // 备注
  createdAt: string;
  updatedAt: string;
}

interface PurchaseItem {
  id: string;
  bambooSpecId: string;   // 竹丝规格ID
  size: string;           // 尺寸，如 "6mm*200mm"
  bundles: number;        // 捆数
  unitPrice: number;      // 单价
  amount: number;         // 金额
  remark: string;         // 备注
}
```

## 4. 页面详细设计

### 4.1 仪表盘 (Dashboard)
**路径**: `/`

**功能**:
- 今日生产概览（消耗竹丝、生产竹筷）
- 本月收支统计
- 待付款订单提醒（进货、出货）
- 近期生产记录
- 库存概览（按尺寸统计竹丝库存、竹筷库存）
- 低库存预警

**图表**:
- 月度生产趋势图（消耗 vs 生产）
- 收支对比图

### 4.2 基础数据模块

#### 4.2.1 竹丝规格管理
**路径**: `/base-data/bamboo-specs`

**功能**:
- 竹丝规格列表（ProTable）
- 新增/编辑/删除规格
- 字段：尺寸、名称、备注、是否启用

#### 4.2.2 竹筷规格管理

**功能**:
- 竹筷规格列表（ProTable）

- 字段：尺寸、名称、备注、是否启用
- 预设常用尺寸快速添加

#### 4.2.3 BOM对应关系
**路径**: `/base-data/bom`

- BOM列表（ProTable）
- 新增/编辑/删除BOM
- 字段：竹筷规格、竹丝规格、每袋耗丝量（捆）

### 4.8 出勤管理模块

#### 4.8.1 月出勤统计表
**路径**: `/attendance`

**功能**:
- 选择年份和月份
- 月出勤表格
  - 行：工人（普通工人）
  - 列：日期（1-31日）
  - 单元格状态：
    - 默认：未上工（灰色背景）
    - 点击1次：全天（绿色背景）
    - 点击2次：半天（黄色背景）
    - 点击3次：未上工（循环）
- 工人工资计算（Modal）
  - 点击工人姓名旁的"计算工资"按钮
  - 显示：
    - 当月出勤天数（全天 + 半天*0.5）
    - 日工资（默认显示工人设置的日工资，可修改）
    - 自动计算：应发工资 = 出勤天数 × 日工资
  - 确认保存工资记录
